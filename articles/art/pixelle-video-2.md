# 阿里 Pixelle-Video：开源AI全自动短视频引擎

> 2025年11月 | 来源: 阿里巴巴 AIDC-AI / GitHub | [原文链接](https://github.com/AIDC-AI/Pixelle-Video)

## 核心要点

阿里巴巴国际数字商务集团（AIDC-AI）实验室开源 **Pixelle-Video**，一个基于 ComfyUI 架构的全自动短视频生成引擎。只需输入一个主题，即可自动完成文案撰写、配图生成、语音解说合成、背景音乐添加和视频合成的全流程，实现"一句话生成完整视频"。

## 关键特性

| 特性 | 说明 |
|------|------|
| **全自动流程** | 从主题输入到成片：文案 → 配图/视频 → 语音解说 → 背景音乐 → 合成输出 |
| **Apache 2.0 开源** | 完全开源，无需商业授权，可自由使用和二次开发 |
| **模块化设计** | 基于 ComfyUI 架构，各模块可独立替换和优化 |
| **图像引导** | 图像的高级语义与视频的时空连贯性深度融合，保持细节保真度 |
| **电影感提示词** | 支持"电影感光效"、"慢动作捕捉"等高级提示词 |

## 社区反响

上线不到半年即获得近 9000 GitHub Star，获得 LMG2025 大会认可。成为 2026 年最受关注的 AI 视频开源工具之一。

## 影视创作影响

Pixelle-Video 极大降低了视频创作的技术门槛，使没有剪辑经验的用户也能快速生成完整的短视频内容。对于内容创作者、自媒体运营者和教育工作者来说，是一个极具实用价值的开源工具。
